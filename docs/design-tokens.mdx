# Design Tokens

Design tokens are the foundational building blocks of our design system. They define the visual design language of the DeutschlandApp, ensuring consistency across all components and pages.

## Location

All design tokens are stored as CSS custom properties in:

**[src/styles/tokens.css](../src/styles/tokens.css)**

This file is imported globally and makes all tokens available throughout the application.

## Exporting from Figma

We use the **[Export Variables](https://www.figma.com/community/plugin/1382804904426179542/export-variables-css-scss-tailwind)** Figma plugin to export design variables from Figma as CSS variable definitions.

To update tokens:
1. Open the Figma design file
2. Run the "Export Variables" plugin
3. Select CSS format
4. Copy the exported CSS variables
5. Update `src/styles/tokens.css` with the new values

## Usage

All tokens are available as CSS custom properties. Use them in your component styles:

```css
.my-component {
  /* Typography */
  font-family: var(--font-family);
  font-size: var(--font-size-lg);
  font-weight: var(--font-weight-semibold);
  line-height: var(--line-height-normal);

  /* Colors */
  color: var(--color-text-primary);
  background-color: var(--color-surface);
  border: 1px solid var(--color-border);

  /* Spacing */
  padding: var(--spacing-4);
  margin-bottom: var(--spacing-6);

  /* Effects */
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-md);
  transition: all var(--transition-base);
}

.my-component:hover {
  border-color: var(--color-border-hover);
}
```
